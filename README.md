# CS2030_Final_Project
*2D Top-Down Horror Game*

Desecrator is a small, atmospheric top-down horror game built for our CS 2030 final project. You explore a haunted graveyard at night, searching for scattered gold while avoiding the creature that hunts you in the dark. Your only defense is your flashlight—use it wisely.

## Installation
1. Clone the repository
2. Open the project in Godot 4.x
3. Press Play to run the game

## Controls
* W A S D — Move
* Mouse — Aim flashlight
* Space / Left Click — Toggle flashlight
* E — Interact (collect gold)
